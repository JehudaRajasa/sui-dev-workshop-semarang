# Sui Dev Workshop Apps

This repository contains the example projects for the 2025 Semarang Sui Developer Workshop.

## Projects

- **hello-world**: A basic introduction to Sui Move.
- **nft-app**: An NFT minting application (Module 4).

## Getting Started

Navigate to each project folder for specific instructions on how to build and run them.

## Prerequisites

- [Sui CLI](https://docs.sui.io/guides/developer/getting-started/sui-install)
- [Node.js](https://nodejs.org/)
- A Node.js package manager (Bun, pnpm or npm)
